Fileuploader bootstrap-fileuploader.js

Examples

Fileuploader is used for muti-file upload support with bootstrap style.

Usage

Enable fileuploader via JavaScript:

$("#element").fileuploader(options);

Options

Options should be passed via JavaScript.

Name type default description
type string single Set file uploader type. single means only could select one file. muti means could select muti-files.
max number -1 It's enabled when type is muti. It's for max number of files. -1 means no limit number.

Events

No events for fileuploader.
