How to use it

You need import the css and js file after original Bootstrap files.

...
<script src="http://code.jquery.com/jquery.js"></script>
<script src="js/bootstrap.min.js"></script>
<link href="css/bootstrap.min.css" rel="stylesheet" media="screen">

<-- Bootstrap Components Plugin -->
<script src="js/bootstrap-component.min.js"></script>
<link href="css/bootstrap-component.min.css" rel="stylesheet" media="screen">
...
