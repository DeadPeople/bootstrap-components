Datepciker bootstrap-datepicker.js

Examples

Datepicker is used for input element which need date type input. When user focus on the element, a datepicker component will popup for user use.

Usage

Enable darklight via JavaScript:

$("#element").datepicker();

When it's called datepicker, it will find the sub element which contains class 'datapicker' as input/output element and find sub element which contains attribute [data-toggle='datepicker'] as trigger:

<div id="datepicker" class="input-append">
	<input class="datepicker" id="appendedInput" type="text" style="width: 140px;">
	<button class="btn" data-toggle="datepicker" href="#"><i class="icon-calendar"></i></button>
</div>

Options

Options should be passed via JavaScript.

Name type default description
target string all all will display date and time selector. timer will display time selector. dater will display date selector.
date Date To set date time. Default date time is now.

Events

Event Description
change It will trigger when datepicker change its value. 
$("#element").change(function(){
	alert("Date Change!");
});
